JavaScript Media Layer Integration

<script src="http://dealermade.co/assets/media-layer/v1/dm.js" 
data-dm-dealership-id="[DEALERMADE DEALERSHIP ID]" 
data-dm-vehicle-vin="[VEHICLE VIN GOES HERE]" async></script>

Required attributes of script tag:

  • src: currently must point to: http://dealermade.co/assets/media-layer/v1/dm.js
  • data-dm-dealership-id: Unique Dealership ID.
  • data-dm-vehicle-vin: Full vin number of vehicle to display pictures for.
