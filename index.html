JavaScript Media Layer Integration

<script src="http://dealermade.co/assets/media-layer/v1/dm.js" 
data-dealership-id="[DEALERMADE DEALERSHIP ID]" 
data-vehicle-vin="[VEHICLE VIN GOES HERE]" async></script>

Required attributes of script tag:

  • src: currently must point to: http://dealermade.co/assets/media-layer/v1/dm.js
  • data-dealership-id: Unique Dealership ID.
  • data-vehicle-vin: Full vin number of vehicle to display pictures for.
