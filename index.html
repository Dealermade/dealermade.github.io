JavaScript Media Layer Integration

<script src="http://dealermade.co/assets/media-layer/v1/dm.js" 
data-dm-dealership-id="[DEALERMADE DEALERSHIP ID]" 
data-dm-vehicle-vin="[VEHICLE VIN]"
data-dm-element-id-to-replace="[ELEMENT ID]"
data-dm-insert-before-element-id="[ELEMENT ID]"
data-dm-element-ids-to-hide="[ELEMENT ID #1]|[ELEMENT ID #2]|[...]"></script>

Required attributes of script tag:

  • src: currently must point to: http://dealermade.co/assets/media-layer/v1/dm.js
  • data-dm-dealership-id: Unique Dealership ID (without brackets).
  • data-dm-vehicle-vin: Full vin number of vehicle to display pictures for (also without brackets).
  • data-dm-element-id-to-replace: One element id to replace with the media layer(not to be used with insert before, below) [optional].
  • data-dm-insert-before-element-id: One element id where the media layer will insert before (not to be used with id to replace, above) [optional].
  • data-dm-element-ids-to-hide: One or many element ids separated by a "|" pipe character that will be hidden when the media layer and the corresponding vehicle pictures are loaded (also without brackets) [optional].

The media layer will remain invisible until pictures have been taken for the vehicle using the dealermade photo studio, also the element ids listed in data-dm-element-ids-to-hide will not be hidden unless the media layer has loaded with pictures.

Example Media Layer Implementation:

<script src="http://dealermade.co/assets/media-layer/v1/dm.js" 
data-dm-dealership-id="team-of-navasota" 
data-dm-vehicle-vin="1FTEW1EF4FFA24461"
data-dm-element-id-to-replace="vpActualContent"
data-dm-element-ids-to-hide="dtlExpand"></script>

http://www.teamofnavasota.com/new/details/1FTEW1EF4FFA24461

Thumbnail Image API

http://dealermade.co/api/1/dealerships/[DEALERMADE DEALERSHIP ID]/vehicles/[VEHICLE VIN]/images/[PICTURE SIZE].jpg?default_image_url=[default_image_url]

Required parameters in the image url:

  • [DEALERMADE DEALERSHIP ID]: Unique Dealership ID (without brackets).
  • [VEHICLE VIN]: Full vin number of vehicle to display pictures for (also without brackets).
  • [PICTURE SIZE]: One of the following standard Dealermade picture size names(also without brackets, full, large, etc.) Ex: thumb.jpg.
    • large 1500w x 1000h
    • medium 800w x 533h
    • main 640w x 427h
    • preview 200w x 133h
    • thumb 160w x 107h
    • mini 60w x 40h
    • micro 25w x 17h
  • [default_image_url]: Url Encoded default image thumbnail for that vehicle. This will be used if pictures have not yet been taken using the Dealermade Photo Studio(also without brackets).

Example Thumbnail Image API Implementation:

<img src="http://dealermade.co/api/1/dealerships/team-of-navasota/vehicles/1FTFW1ET3DKD06282/images/thumb.jpg?default_image_url=http%3A%2F%2Ftagoreimg.d2fs.net%2Fimg%2Fused%2Fford%2Ff-150%2F1ftfw1et3dkd06282%2F3777d222-6258-4cf3-b694-6d6acd014c01.jpeg%3Fw%3D160%26h%3D120"/>
